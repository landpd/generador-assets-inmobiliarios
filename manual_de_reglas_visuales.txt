### REGLAS GLOBALES DE ESTRUCTURA Y NAVEGACIÓN
Estas reglas aplican transversalmente a casi todas las piezas para asegurar la legibilidad y la interacción del usuario.

1.  **Jerarquía Tipográfica Estricta:**
    *   **Titulares (H1):** Deben ocupar al menos un 25-30% del espacio visual. Trazos ultragruesos (Black/Bold) o tipografías condensadas para generar impacto.
    *   **Cuerpo de texto (Body):** Uso de pesos regulares o ligeros. El tamaño debe ser al menos un 60% más pequeño que el titular.
    *   **Regla de contraste:** Nunca usar el mismo peso tipográfico para el título y el cuerpo.
2.  **Indicadores de Carrusel (Navegación):**
    *   Es imperativo el uso de **flechas minimalistas** (líneas finas, sin relleno) ubicadas en la mitad derecha o en la esquina inferior derecha para incentivar el "swipe" (deslizar).
3.  **Uso del Espacio Negativo (Aire):**
    *   Los márgenes perimetrales deben ser amplios (mínimo un 10% del lienzo). El texto nunca debe tocar los bordes de la imagen.
    *   Agrupar elementos lógicos (Título + Subtítulo) y separarlos visualmente de las imágenes o iconos mediante espacio vacío.

### ARQUETIPOS DE PLANTILLAS (Estructuras de Diseño)
Basado en las imágenes, se pueden desarrollar **4 arquetipos de plantillas** distintos.

#### Plantilla Tipo 1: Estilo Editorial / Collage (Ref. "La Jornada", "No todo el mundo")
*Estructura ideal para noticias, citas (quotes) y finanzas.*
*   **Fondo:** Uso obligatorio de texturas sutiles (papel arrugado, papel reciclado con porosidad, ruido/grano). No usar blancos puros.
*   **Composición Asimétrica:** Dividir la pantalla vertical u horizontalmente.
    *   *Opción A:* Texto gigante a la izquierda, bloque de imagen (con bordes curvos o marcos) a la derecha.
    *   *Opción B:* Bloque de color sólido central para el texto, imagen silueteada en la base.
*   **Silueteado (Cut-outs):** Uso de personajes u objetos recortados (sin fondo) que se superponen a los bloques de color o al texto de fondo, creando un efecto 3D y de profundidad.
*   **Bloques de Contención:** Utilizar rectángulos de colores sólidos (rojo, ocre, azul) para enmarcar frases clave o actuar como base para el texto.

#### Plantilla Tipo 2: Corporativo Minimalista de Listas (Ref. "Colliers", "1. Ubicación")
*Estructura ideal para datos duros, pasos, factores y contenido B2B.*
*   **El Número como Héroe:** Los números de las listas (1, 2, 3...) deben ser gigantes, ocupando un lugar protagónico (esquina superior izquierda o alineados a la izquierda).
*   **Fondos:** Dos variantes:
    *   Color sólido vibrante (ej. Azul corporativo) en el 100% del lienzo.
    *   Fondo blanco puro o gris muy claro.
*   **Cajas Flotantes (Cards):** Si el fondo es una fotografía, el texto debe ir dentro de una caja de color sólido con opacidad al 100% (sin transparencias) que cubra el 60-70% del diseño para asegurar legibilidad.
*   **Iconografía:** Uso de iconos de línea (line-art) del mismo color que el texto principal, ubicados estratégicamente para equilibrar el peso visual del texto.

#### Plantilla Tipo 3: Cinematográfico / Cultural (Ref. "1984", "Filmmakerlife", "Películas que parecen poesía")
*Estructura ideal para arte, reseñas, cine y literatura.*
*   **Fondo a Sangre (Full Bleed):** La fotografía o ilustración debe ocupar el 100% del lienzo.
*   **Tratamiento de Imagen:** Uso intenso de ruido visual (grano de película), viñetas oscuras en los bordes y bajo contraste.
*   **Texto sobre Imagen:** El texto principal suele ubicarse centrado en la parte superior o inferior. Se deben usar tipografías que evoquen poesía o historia (Serif elegantes o Sans-serif muy pesadas, estilo cartelera).
*   **Recursos Geométricos Inserción:** Si hay dos elementos visuales, uno es el fondo y el otro se inserta dentro de una forma geométrica (ej. un círculo perfecto en una esquina superior) enmarcado con una línea delgada.

#### Plantilla Tipo 4: Promocional de Producto (Ref. "Giant")
*Estructura ideal para e-commerce y lanzamientos.*
*   **Fondo Monocromático:** Color plano vibrante (ej. Verde menta) que contraste con el producto.
*   **Tipografía Invasiva:** El titular principal ("NO ESPERES...") está alineado a la izquierda, en mayúsculas, y se come gran parte del espacio vertical.
*   **Marca de Agua / Superposición:** Uso de elementos gráficos gigantes (ej. el símbolo "%") en el fondo, con baja opacidad o el mismo color del fondo pero con diferente tono, actuando como textura.
*   **El Producto:** Recortado perfectamente, escalado a gran tamaño, interrumpiendo o sobreponiéndose ligeramente a los elementos gráficos de fondo.

### PALETA DE COLOR Y TEXTURAS (Reglas de Implementación)
*   **Contraste Alto:** Textos oscuros (Azul marino, Negro, Gris carbón) sobre fondos claros, o Textos blancos sobre fondos oscuros o saturados (Azul cobalto, Rojo, Negro). Evitar grises medios para textos.
*   **Paleta Limitada:** Cada diseño no debe utilizar más de 3 colores principales (Ej: Blanco, Azul Corporativo y un color de acento para la flecha o líneas).
*   **Texturas como elemento gráfico:** Incorporar una biblioteca de texturas (papel, grano fotográfico) para aplicarlas en modo de fusión (Multiplicar o Superponer) con opacidad entre el 10% y el 30% en las plantillas tipo Editorial/Cultural.
